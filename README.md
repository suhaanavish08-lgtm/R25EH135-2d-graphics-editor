# 2D Graphics Editor in C

A menu-driven 2D Graphics Editor developed in C using a 2D character array as the drawing canvas.  
The project allows users to create and manipulate basic graphical objects directly in the terminal.

---

## 📌 Project Description

This project simulates a simple graphics editor where the drawing area is represented using a 2D array of characters.  
The canvas is initially filled with `_` (underscore) characters, while graphical objects are drawn using `*` (asterisk).

The application is completely menu-driven and supports drawing, deleting, modifying, and displaying shapes.
